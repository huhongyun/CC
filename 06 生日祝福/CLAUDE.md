# 生日祝福诗项目

## 项目目标

为朋友生成个性化生日祝福藏头诗，以「姓名+生日快乐」为藏头，支持保存到 Get笔记。

## 文件结构

```
06 生日祝福/
├── CLAUDE.md              # 本文件
└── 生日祝福需求.txt         # 原始需求文档
```

Skill 本体位于 `~/.claude/skills/birthday-poem/SKILL.md`（用户级，不在项目 repo 内）。

## 使用方式

在 Claude Code 中输入 `/birthday-poem` 或说「生日祝福」「写生日祝福」触发。

流程：收集信息（姓名、性别、关系、年龄段、祝福重点、风格）→ 生成藏头诗 → 自动迭代优化 3 次 → 可选保存到 Get笔记。

## 藏头规则

- 2字姓名 → 6句（每句一个藏头字）
- 3字姓名 → 7句（每句一个藏头字，不可将两个字放在同一行）

## Get笔记集成

- 保存到「生日祝福」知识库（ID: `eYx4y5xY`）
- 标签：`生日祝福`、`诗歌`
- 标题格式：`生日祝福诗 - [姓名] - [YYYY-MM-DD]`
- **保存必须用 Python**（curl 在 Windows bash 下中文乱码）
